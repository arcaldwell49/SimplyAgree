# reps

A fake data set of a agreement study where both measures have
replicates.

The data set published in the original Bland & Altman paper on
agreement.

## Usage

``` r
reps

ba1986
```

## Format

A data frame with 20 rows with 3 variables

- id:

  Subject identifier

- x:

  X measurement

- y:

  Y measurement

A data frame with 17 rows with 5 variables

- id:

  Subject identifier

- wright1:

  PERF measurement \#1 using Wright device

- wright2:

  PERF measurement \#2 using Wright device

- mini1:

  PERF measurement \#1 using Mini device

- mini2:

  PERF measurement \#2 using Mini device

## References

Bland, J. M., & Altman, D. (1986). Statistical methods for assessing
agreement between two methods of clinical measurement. The Lancet,
327(8476), 307-310.
